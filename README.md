# Sai Diagnostic Lab Management System

A premium MERN stack application for managing diagnostic lab reports and slips.

## 🚀 Live Demo
- **Frontend:** [https://doctorslip-frontend-7igmpp2ak-raghavkainses-projects.vercel.app](https://doctorslip-frontend-7igmpp2ak-raghavkainses-projects.vercel.app)
- **Backend API:** [https://doctorslip-backend-ujktc-raghavkainses-projects.vercel.app](https://doctorslip-backend-ujktc-raghavkainses-projects.vercel.app)

## ✨ Features
- **Responsive Design:** Works perfectly on Mobile, Tablet, and Desktop.
- **Report Generation:** Create professional A4 reports with automated range checking (High/Low).
- **Slip Generation:** Quick billing slips with WhatsApp sharing.
- **History Management:** View and download old reports/slips.
- **Test Master:** Manage test templates and parameters.
- **Address Field:** Optional address field for patients.

## 🛠 Tech Stack
- **Frontend:** React, Tailwind CSS, Lucide Icons.
- **Backend:** Node.js, Express, MongoDB (Mongoose).
- **Deployment:** Vercel.

## ⚙️ Setup
1. Clone the repository.
2. Install dependencies for both `frontend` and `backend`.
3. Set `MONGO_URI` in backend `.env`.
4. Run `npm run dev` in both folders.
